# Autoobenz Site

نسخة محلية قابلة للتشغيل من موقع أوتو بنز باستخدام HTML/CSS/JavaScript فقط.

## التشغيل

```powershell
npm start
```

ثم افتح:

```text
http://localhost:4173
```

## الملفات المهمة

- `index.html`: هيكل الموقع.
- `styles.css`: التصميم.
- `app.js`: المنتجات، الفلاتر، صفحات المنتج، والسلة.
- `assets/data`: بيانات المنتجات والتصنيفات والماركات.
- `assets/images`: صور الموقع والمنتجات.
